\# RAG PDF Project



\## Overview

A Retrieval-Augmented Generation (RAG) project using:

\- LangChain

\- FAISS

\- HuggingFace Embeddings

\- Mistral AI



\## Features

\- Load PDF files

\- Split documents into chunks

\- Generate embeddings

\- Store vectors in FAISS



\## Installation



pip install -r requirements.txt



\## Run



python create\_database.py
